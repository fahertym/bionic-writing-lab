# Bionic Writing Lab Operating Notes

This repository is a Markdown-first static publishing system for many kinds of writing, not a single manuscript repo.

Working rules:

- Markdown source files are canonical.
- Publication manifests define identity, routing, and outputs.
- Do not center the architecture on any one existing project.
- Keep books, essays, poems, posts, pamphlets, collections, and series distinct where that helps clarity.
- Prefer simple static generation and readable Python scripts over framework or backend sprawl.
